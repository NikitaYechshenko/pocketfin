# Copilot instructions (Asset Tracker)

## Big picture
- Backend is a **FastAPI** app with a simple layered module pattern: **router → service → repository → SQLAlchemy model**.
- DB access is **sync SQLAlchemy** (`SessionLocal` + `create_engine`), configured from `.env` via Pydantic Settings.
- Migrations are **Alembic**, wired to read the DB URL from app settings (not from `alembic.ini`).

## Key entrypoints
- API app creation + router registration: `app/main.py` (includes `app.modules.users.router`).
- Settings: `app/core/config.py` (`DATABASE_URL`, `SECRET_KEY` from `.env`).
- DB session dependency: `app/core/database.py` (`get_db()` yields a `Session`).
- Health check: `app/core/health.py` (фоновая проверка БД каждые 2 минуты).
- Database health endpoint: `GET /health` для мониторинга соединения.

## Module conventions (example: users)
- Router: `app/modules/users/router.py`
  - Endpoints live here and should be thin; use `Depends(get_db)` to get a DB session.
  - All endpoints log request start, success, and errors with appropriate levels (info/warning/error).
- Service: `app/modules/users/service.py`
  - Business logic (registration/auth) lives here; raises `HTTPException` for domain errors.
- Repository: `app/modules/users/repository.py`
  - DB queries and persistence; note there are **backward-compatible aliases** (`get_by_email` → `get_user_by_email`, `create` → `create_user`).
- SQLAlchemy models: `app/modules/users/models.py` (declares `Base` + `__tablename__ = "users"`).
- Pydantic schemas: `app/modules/users/schemas.py` (responses use `Config.from_attributes = True`).

## Auth/token notes
- Token creation is implemented in `app/core/security.py` (HS256 JWT).
- `ACCESS_TOKEN_EXPIRE_MINUTES` is currently **1 minute**.
- The sample frontend sends `Authorization: Bearer <token>` for `/users/`, but the backend route is not yet protected.

## Logging & Monitoring
- Logging configured in `app/main.py` with format: `%(asctime)s - %(name)s - %(levelname)s - %(message)s`
- Database health check runs automatically every **2 minutes** via `start_health_check(interval=120)`
- All database operations have connection monitoring via `pool_pre_ping=True` and `pool_recycle=3600`
- Router endpoints log: request start (info), success (info), domain errors (warning), critical errors (error)

## Migrations (Alembic)
- Alembic config is in `alembic/env.py` and forces `sqlalchemy.url` from `.env`.
- For `--autogenerate` to see new models, import them in `alembic/env.py` (currently imports `app.modules.users.models`).

## Local dev workflow (Windows)
- Start Postgres: `docker compose up -d` (see `docker-compose.yml`, host port **5433** → container **5432**).
- Configure `.env` (file is gitignored):
  - `DATABASE_URL=postgresql+psycopg://postgres:postgres@localhost:5433/gat_db`
  - `SECRET_KEY=...`
- Install deps: `pip install -r requirements.txt`
- Apply migrations: `alembic upgrade head`
- Run API: `uvicorn app.main:app --reload`

## Frontend integration
- `frontend/login.html` calls `http://127.0.0.1:8000`:
  - `POST /users/register`, `POST /users/login`, `GET /users/?skip=0&limit=100`

## Dependency management
- `requirements.txt` is generated by `pip-compile` from `requirements.in`.
- Update `requirements.in`, then regenerate (example): `pip-compile requirements.in`

import asyncio
from logging.config import fileConfig

from alembic import context
from sqlalchemy import pool
from sqlalchemy.engine import Connection
from sqlalchemy.ext.asyncio import async_engine_from_config, create_async_engine

from app.core.config import settings
from app.core.database import Base
# Импорт моделей ОБЯЗАТЕЛЕН, чтобы Alembic их увидел
from app.modules.users import models  # noqa

# Alembic Config object
config = context.config

# Setup logging
if config.config_file_name is not None:
    fileConfig(config.config_file_name)

# Set DB URL
config.set_main_option("sqlalchemy.url", settings.DATABASE_URL)

target_metadata = Base.metadata

def run_migrations_offline() -> None:
    """Run migrations in 'offline' mode."""
    url = settings.DATABASE_URL
    context.configure(
        url=url,
        target_metadata=target_metadata,
        literal_binds=True,
        dialect_opts={"paramstyle": "named"},
        compare_type=True,
    )

    with context.begin_transaction():
        context.run_migrations()


def run_migrations_online() -> None:
    """Run migrations in 'online' mode using AsyncEngine."""

    # Создаем движок
    connectable = create_async_engine(
        settings.DATABASE_URL,
        poolclass=pool.NullPool,
    )

    async def do_run_migrations():
        # !!! ИСПРАВЛЕНИЕ ЗДЕСЬ !!!
        # Мы используем 'async with', чтобы реально открыть соединение
        async with connectable.connect() as connection:
            await connection.run_sync(do_run_migrations_sync)
        
        # Хорошая практика: закрыть движок после использования
        await connectable.dispose()

    def do_run_migrations_sync(connection: Connection):
        context.configure(
            connection=connection,
            target_metadata=target_metadata,
            compare_type=True, # Важно для отслеживания изменений типов колонок
        )

        with context.begin_transaction():
            context.run_migrations()

    # Запускаем асинхронную функцию
    asyncio.run(do_run_migrations())


if context.is_offline_mode():
    run_migrations_offline()
else:
    run_migrations_online()

from typing import Optional

from fastapi import Depends, Request
from fastapi_users import BaseUserManager, FastAPIUsers, IntegerIDMixin
from fastapi_users.authentication import (
    AuthenticationBackend,
    BearerTransport,
    JWTStrategy,
)
from fastapi_users.db import SQLAlchemyUserDatabase
from sqlalchemy.ext.asyncio import AsyncSession

from app.core.config import settings
from app.core.database import get_async_session
from app.modules.users.models import User

# 1. Настройка БД для библиотеки
async def get_user_db(session: AsyncSession = Depends(get_async_session)):
    yield SQLAlchemyUserDatabase(session, User)

# 2. Менеджер пользователей (бизнес-логика)
class UserManager(IntegerIDMixin, BaseUserManager[User, int]):
    reset_password_token_secret = settings.SECRET_KEY
    verification_token_secret = settings.SECRET_KEY

    async def on_after_register(self, user: User, request: Optional[Request] = None):
        print(f"User {user.id} has registered.")

async def get_user_manager(user_db=Depends(get_user_db)):
    yield UserManager(user_db)

# 3. Настройка Аутентификации (JWT)
# Transport = Bearer (токен в заголовке Authorization: Bearer <token>)
bearer_transport = BearerTransport(tokenUrl="auth/jwt/login")

def get_jwt_strategy() -> JWTStrategy:
    return JWTStrategy(secret=settings.SECRET_KEY, lifetime_seconds=3600)

auth_backend = AuthenticationBackend(
    name="jwt",
    transport=bearer_transport,
    get_strategy=get_jwt_strategy,
)

# 4. Главный объект FastAPIUsers (связывает всё воедино)
fastapi_users = FastAPIUsers[User, int](
    get_user_manager,
    [auth_backend],
)

# 5. Хелпер для получения текущего юзера (замена нашей старой функции)
current_active_user = fastapi_users.current_user(active=True)

        <input type="email" id="loginEmail" placeholder="Email" />
        <input type="password" id="loginPass" placeholder="Пароль" />
        <button onclick="login()">Войти</button>
        <button class="secondary" onclick="toggleScreen('register')">Нет аккаунта? Регистрация</button>
    </div>

    <div id="registerForm" class="hidden">
        <input type="email" id="regEmail" placeholder="Email" />
        <input type="password" id="regPass" placeholder="Пароль (минимум 3 символа)" />
        <button onclick="register()">Зарегистрироваться</button>
        <button class="secondary" onclick="toggleScreen('login')">Назад ко входу</button>
    </div>

    <div id="profileArea" class="hidden">
        <p>Вы вошли как:</p>
        <div id="userInfo" style="font-weight: bold; margin-bottom: 10px;"></div>
        <button onclick="getMe()">Обновить данные (Get Me)</button>
        <button class="secondary" style="background: #dc3545;" onclick="logout()">Выйти</button>
        
        <p style="margin-top: 15px; font-size: 0.9rem;">Ответ сервера:</p>
        <pre id="serverData">...</pre>

<script>
    const API_URL = "http://127.0.0.1:8000";

    // Управление экранами
    function toggleScreen(screen) {
        document.getElementById('errorMsg').innerText = "";
        document.getElementById('loginForm').classList.add('hidden');
        document.getElementById('registerForm').classList.add('hidden');
        document.getElementById('profileArea').classList.add('hidden');
        
        if(screen === 'login') {
            document.getElementById('title').innerText = "Вход";
            document.getElementById('loginForm').classList.remove('hidden');
        } else if (screen === 'register') {
            document.getElementById('title').innerText = "Регистрация";
            document.getElementById('registerForm').classList.remove('hidden');
        } else if (screen === 'profile') {
            document.getElementById('title').innerText = "Профиль";
            document.getElementById('profileArea').classList.remove('hidden');
        }
    }

    // 1. РЕГИСТРАЦИЯ (Отправляем JSON)
    async function register() {
        const email = document.getElementById('regEmail').value;
        const password = document.getElementById('regPass').value;
        const err = document.getElementById('errorMsg');

        try {
            const res = await fetch(`${API_URL}/auth/register`, {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({ email, password })
            });
            
            if (res.ok) {
                alert("Регистрация успешна! Теперь войдите.");
                toggleScreen('login');
                document.getElementById('loginEmail').value = email;
            } else {
                const data = await res.json();
                err.innerText = "Ошибка: " + JSON.stringify(data.detail);
            }
        } catch (e) {
            err.innerText = "Ошибка соединения: " + e;
        }
    }

    // 2. ВХОД (Отправляем Form Data - требование OAuth2)
    async function login() {
        const email = document.getElementById('loginEmail').value;
        const password = document.getElementById('loginPass').value;
        const err = document.getElementById('errorMsg');

        // FormData для x-www-form-urlencoded
        const formData = new URLSearchParams();
        formData.append('username', email); // Важно: поле username, а не email
        formData.append('password', password);

        try {
            const res = await fetch(`${API_URL}/auth/jwt/login`, {
                method: 'POST',
                headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
                body: formData
            });

            if (res.ok) {
                const data = await res.json();
                localStorage.setItem('token', data.access_token);
                toggleScreen('profile');
                getMe(); // Сразу загружаем данные юзера
            } else {
                err.innerText = "Неверный логин или пароль";
            }
        } catch (e) {
            err.innerText = "Ошибка соединения";
        }
    }

    // 3. ПОЛУЧЕНИЕ ДАННЫХ (Защищенный роут)
    async function getMe() {
        const token = localStorage.getItem('token');
        if (!token) return logout();

        try {
            const res = await fetch(`${API_URL}/users/me`, {
                method: 'GET',
                headers: { 'Authorization': `Bearer ${token}` }
            });

            if (res.ok) {
                const user = await res.json();
                document.getElementById('userInfo').innerText = user.email;
                document.getElementById('serverData').innerText = JSON.stringify(user, null, 2);
            } else {
                logout(); // Токен протух
            }
        } catch (e) {
            console.error(e);
        }
    }

    function logout() {
        localStorage.removeItem('token');
        toggleScreen('login');
    }

    // Проверка при загрузке
    if (localStorage.getItem('token')) {
        toggleScreen('profile');
        getMe();
    }
</script>
